# Installation

##  _Termux_

```cmd

> pkg update && pkg upgrade

> pkg install git -y

> pkg install nodejs -y

> pkg install ffmpeg -y

```

## _Cloning_

```cmd

> git clone https://github.com/SofyanAMV09 /LoL-Bot

> cd LoL-Bot

```

## _Install_

```cmd

> npm i

```

## _Run the Lol-Bot_

```cmd

> npm start

```
